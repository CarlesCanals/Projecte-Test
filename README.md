# Projecte-Test

## Descripció
Projecte-Test és una aplicació web personal desenvolupada per Carles Canals. Permet gestionar el perfil d'usuari, registrar-se, iniciar sessió i consultar informació sobre els projectes i experiència professional del desenvolupador.

## Funcionalitats
- Portada atractiva amb informació personal i enllaços a les funcionalitats principals.
- Formulari d'inici de sessió per accedir a l'espai personal.
- Formulari de registre per crear un nou compte d'usuari.
- Navegació clara entre les diferents pàgines.

## Tecnologies
- HTML5
- Bootstrap 5 per al disseny i estils moderns
